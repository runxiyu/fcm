Website Hosting

This Website started in January, 2022. It's a work in progress. If you would like to contribute, please send them to fcm@andrewyu.org.

This Website, along with the Libre Society project, are looking for a domain name to host on, since it doesn't make sense for this to be a subdomain of Andrew's personal Website.

Please send general inquiries, corrections and suggestions to the fcm@andrewyu.org mailing list. Ask in it to join! Currently there is no community IRC channel, just use #fsf on Libera.Chat (not sponsored).
