The Origin of this Movement

The Free Software Foundation is a nonprofit with a worldwide mission to promote computer user freedom. However, as we've discussed in an email thread via LibrePlanet-discuss, merely having free software doesn't give us the freedom we deserve.

In fact, access to schematics, boardviews, datasheets and any other documentation is *critical* to software freedom.

Leah Rowe
